# 한국 업체 중복 탐지 – 구조 설명

## 1. 입력 파일 구조 (FOOD 중복업체.xlsx)

Excel/CSV 파일은 **아래 3개 컬럼**이 있으면 됩니다.

| 컬럼명       | 설명           | 예시 |
|-------------|----------------|------|
| **공급업체코** | 업체 고유 코드 (숫자 등) | 337892, 337895 |
| **공급업체명** | 업체명 (한글)   | (주)오메가오토메이션 |
| **주소**       | 전체 주소 (한글) | 경기도 부천시 소사구 양지로 237 522~526호 (옥길동, 광양프런티…) |

- 컬럼명은 **공급업체코드**로 되어 있어도 인식합니다.
- 그 외 컬럼이 더 있어도 무시되고, 위 3개만 사용합니다.

---

## 2. 처리 흐름 (구조)

```
[FOOD 중복업체.xlsx]
        │
        ▼
┌─────────────────────────────────────┐
│  src/data_loader_korean.py          │
│  • Excel 로드                        │
│  • 컬럼 매핑: 공급업체코 → 공급업체코드 │
│  • Land = 'KR', 주소에서 시/군/구 추출  │
└─────────────────┬───────────────────┘
                  ▼
┌─────────────────────────────────────┐
│  src/duplicate_detector_hybrid.py    │
│  • Blocking + 임베딩으로 후보 추출   │
│  • STEP1: 후보 그룹 (Cheap Gate)     │
│  • STEP2: 최종 중복 그룹 (Heavy Judge)│
└─────────────────┬───────────────────┘
                  ▼
┌─────────────────────────────────────┐
│  src/merger.py                      │
│  • 그룹번호_STEP1, 그룹번호_STEP2 부여 │
│  • 매칭 정보(confidence 등) 추가      │
└─────────────────┬───────────────────┘
                  ▼
[output/korean_vendor_merged_hybrid_YYYYMMDD_HHMMSS.xlsx]
```

## 3. 출력 파일 구조 (결과 Excel)

결과 Excel에는 대략 아래 컬럼들이 붙습니다.

| 컬럼               | 설명 |
|--------------------|------|
| **vendor_id**      | 원본 행 번호(인덱스) |
| **old_group**      | (미사용 시 빈칸) |
| **그룹번호_STEP1** | 후보 그룹 번호 (추가 검토용) |
| **그룹번호_STEP2** | 최종 중복 그룹 번호 (같은 번호 = 동일 업체로 판단) |
| **match_type**     | 매칭 유형 (예: NORMALIZED_NAME_MATCH) |
| **match_reason**   | 매칭 이유 (한 줄 설명) |
| **confidence**     | 신뢰도 (0~1) |
| **공급업체코드**   | 원본 공급업체코 |
| **공급업체명**     | 원본 공급업체명 |
| **Land**           | KR |
| **CITY1**          | 주소에서 추출한 시/군/구 |
| **STR_SUPPL1**     | 전체 주소 |
| **_duplicate_count** | 해당 행이 속한 STEP2 그룹의 행 수 |

- **그룹번호_STEP2**가 같은 행들 = 같은 업체로 판단된 중복 그룹입니다.
- **그룹번호_STEP2**가 비어 있으면 "중복 아님"으로 분류된 행입니다.

---

## 4. 실행 방법

**FOOD 중복업체.xlsx**를 프로젝트 폴더(`duplCkSupply`)에 넣은 뒤:

```bash
python run_hybrid_korean.py
```

다른 파일을 쓰려면:

```bash
python run_hybrid_korean.py "다른파일.xlsx"
```

- 로그: `output/process_log_hybrid_korean.txt`
- 체크포인트: `output/checkpoint_ann_korean.pkl`
- 결과: `output/korean_vendor_merged_hybrid_YYYYMMDD_HHMMSS.xlsx`
